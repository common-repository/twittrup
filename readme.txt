Updates Twitter when you create a new blog post utilizing an shortener service of your choice.

== Description ==
This plugin updates your twitter status with the latest post that you've made.  It also has the option of including a link back to your post so people can easily click on it to view the post.
The link provided in the tweet can be shorted via a service of your choice: tinyurl.com, is.gd, bit.ly or snurl.com

If you wan to stay up with me, you can <a href="http://twitter.com/marcobischoff">follow me</a>.
Enjoy!

== Installation ==

1. Extract twittrup.zip
1. Upload `twittrup` to the `/wp-content/plugins/` directory
2. activate the plugin through the `Plugins` menu in WordPress
3. Go to Manage -> Twittrup, select the options you want to use, then verify your Twitter login details
4. That's it!  You're all set and ready to update your Twitter status with post updates
